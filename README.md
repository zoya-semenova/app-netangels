## Web Server

Сервер отображается по 80 порту на домене `http://localhost`.


#### Connect via SSH

Команда входа в контейнер с сайтом.
```shell
docker-compose exec webserver bash
```

## PHP

Все настройки конфигурации осуществляются в файле `.env`file. 

## phpMyAdmin

phpMyAdmin доступен по `8080` порту. Логин и пароль можно поменять в файле конфигурации `.env`

http://localhost:8080/  
username: root  
password: tiger

Если размер базы большой и загрузить через phpMyAdmin не получается, то это можно сделать командо
```
docker exec -i lamp-database mysql -uroot -ptiger stroytek_1 < /home/lamer/Загрузки/stroytek.sql
```

## Redis

Доступен по `6379` порту, имя хоста `redis`

## Maildev

Для проверки, отправки писем настрое почтовый клиент. Доступен данный клиент на `8282` порту.
Доступ можно получить по URI:

http://localhost:8282/ 

## Начало работы с контейнерами

Так же для автоматизации процесса запуска и остановки контейнера создан баш скрипт.
В данном скрипте нужно сделать единственную настройку. Указать путь до папки с проектами.

Запустить контейнер
```shell
sh devdomain.sh up osnova
```
Отключить контейнер
```shell
sh devdomain.sh down
```
